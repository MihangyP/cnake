# Cnake
This is a simple snake game written in pure C.

## How to build and run the program?
To build:
```console
$ cc nob.c -o nob
$ ./nob
```
To run:
```console
$ ./build/cnake
```

### Libraries used:
- Xlib - for graphics rendering
- miniaudio - for sound effects
- NoBuild (or nob) - to build the project
- FreeType - for font rendering
